# Dashboard Kota Bandung Tahun 2025 🏙️

An interactive web dashboard built with Streamlit and Plotly to visualize the demographic, employment, and spatial data of Bandung City. 

**Live Demo:** [View App Here](https://dashboard-kotabandung.streamlit.app/)

## About The Project
I built this dashboard to transform raw statistical data from BPS (Badan Pusat Statistik) into actionable insights. Rather than just presenting plain numbers, this tool is designed to help users understand the city's population structure, workforce distribution, and potential business opportunities based on regional density.

## Key Features
* **Demographics & Education:** Features an interactive population pyramid, gender ratio analysis, and education attainment breakdowns to spot the city's demographic dividend.
* **Labor Force Profiling:** Visualizes the workforce distribution based on industry sectors and employment status (formal vs. informal).
* **Spatial Business Analysis:** Compares population density against the number of F&B businesses (restaurants) across different sub-districts. This helps identify "unmet demand" areas—highly populated zones with low commercial facilities.
* **Modern UI:** A clean, dark-themed enterprise interface optimized for data readability and presentation.

## Tech Stack
* **Language:** Python
* **Framework:** Streamlit
* **Data Manipulation:** Pandas, OpenPyXL
* **Data Visualization:** Plotly
* **Data Source:** BPS: Kota Bandung Dalam Angka 2026
